# Vrstvy TCP/IP
---
## 4V
- plní stejnou fc jako 7, 6, 5 ISO/OSI
- Služby - el. pošta, přenos souborů, sdílení souborů, vzdálené přihlašování, správa sítě, ...
- obsahuje napojení na aplikace a OS
## 3V (= ISO/OSI)
- Služby/protokoly
	- TCP – *Transmission Control Protocol*
		- spolehlivá služba = ověření přenosové cesty, nastavení 1 cesty, kontrola správnosti dat po přenosu
	- UDP – *User Data Protocol*
		- rychlý negarantovaný přenos dat (různé přenosové cesty pro PACKETY, není kontrola po přenosu) = rychlý, "nespolehlivý" (zvuk, video)
